    "\n",
    "### Agent Workflow\n",

    "2.  *Explainer Agent:* Takes the retrieved context and generates a summarized explanation.\n",
    "3.  *Planner Agent:* Creates a study schedule based on the requested topic.\n",
    "4.  *QuizBot Agent:* Generates questions to test the user's understanding and calculates a score.\n",
    "\n",

    "## Requirements Implemented\n",
    "This project covers 5 key concepts from the course curriculum:\n",
    "\n",

    "\n",
    "*2. Sessions & Memory*\n",
    "The system includes a MemoryBank class. This allows the agents to store conversation history and recall it later using a basic keyword search with recency bias.\n",

    "*3. Context Engineering (RAG)*\n",
    "Instead of sending raw queries to the LLM, the system retrieves relevant data from SimpleVectorStore first to provide context for the answer.\n",
